
# Hi, I'm Edward Maina 👋

Full-stack software developer and second-year Computing student at **Strathmore University** (School of Computing and Engineering Sciences). I build production systems and have a deep interest in motorsport engineering and data analysis.

## What I'm Working On

🏫 **School-Marks-System** — A production school management system serving Santa Ana Calm Waters Academy. Built with Spring Boot, React, and PostgreSQL, deployed on Railway + Vercel. Handles marks entry, PDF report cards, admin analytics, and teacher management for real users.

🏎️ **F1 Telemetry Analysis** — Race data analysis using Python and FastF1. Lap time comparisons, tyre degradation modeling with linear regression, and multi-driver performance visualizations from real Formula 1 session data.

🔧 **F1 Team Management API** — A REST API modeling F1 team operations built with Spring Boot, JPA, and PostgreSQL.

## Tech Stack

**Languages:** Java, JavaScript, Python, SQL  
**Backend:** Spring Boot, Spring Security, JPA/Hibernate, REST APIs  
**Frontend:** React, Vite, HTML/CSS  
**Data:** PostgreSQL, FastF1, NumPy, Matplotlib  
**Tools:** Git, Railway, Vercel, Maven, npm

## Where I'm Headed

My long-term goal is to work in motorsport engineering — combining software, data, and systems thinking to contribute to professional racing teams. I'm currently building the technical foundation and portfolio to make that happen.
